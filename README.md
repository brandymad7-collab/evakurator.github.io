# Личный кабинет службы эвакуации

Веб-приложение личного кабинета для клиентов службы эвакуации автомобилей.

## Функциональность
- Просмотр истории поездок
- Управление данными клиента
- Информация об автомобиле
- Служба поддержки

## Использование
Приложение готово для встраивания в Tilda через iframe.
