# Evakurator - Сервис эвакуации автомобилей

## Описание
Веб-приложение для управления службой эвакуации автомобилей

## Структура проекта
- `index.html` - Главная страница выбора роли
- `client.html` - Личный кабинет клиента
- `dispatcher.html` - Панель управления диспетчера

## Используемые технологии
- HTML5, CSS3, JavaScript
- Bootstrap 5
- Firebase (Authentication, Firestore)
- Font Awesome

## Настройка Firebase
1. Создайте проект в [Firebase Console](https://console.firebase.google.com)
2. Включите Authentication (Email/Password)
3. Создайте базу данных Firestore
4. Обновите конфигурацию в файлах
